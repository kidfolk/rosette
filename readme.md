### Rosette ###

Android Graphics Demonstration

# Install #

Update local.properties to point to Android SDK install location.

```gradlew installDebug```

# Uninstall #

```gradlew uninstallDebug```

# Code Quality Checks #

```gradlew findbugs```
```gradlew check``` (PMD, Checkstyle)
